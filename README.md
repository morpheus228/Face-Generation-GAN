# Face-Generation-GAN

##For model's fitting:
1. Download zip with training images (https://drive.google.com/uc?id=0BxYys69jI14kYVM3aVhKS1VhRUk).
2. Unpack it.
3. Run next comamnd "python fit.py -<path with images> -<path for saving models>"
